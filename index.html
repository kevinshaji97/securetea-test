About

The OWASP SecureTea Project is an application designed to help secure a person's laptop.

  • OBJECTIVE

    To alert the user via various communication mechanism, whenever The Computer / Server had been accessed by someone / attacker. And also it can be used to monitor your system & SecureTea firewall as Defense.

  • FUNCTIONING

    With the help of IOT notify users, whenever someone accesses their computer / server. This application uses the touchpad/mouse/wireless mouse to determine activity and is developed in Python and tested on various machines.

INSTALLATION

Github

Installing from GitHub involves the following steps:

git clone https://github.com/OWASP/SecureTea-Project.git

Install SecureTea package

sudo python setup.py install

Install python dependencies

pip install -r requirements.txt

PyPi

You can install SecureTea from PyPi package manager.

pip install securetea

Please make sure all dependencies are installed if this fails.

ZIP

Download the Zip Unzip using:

unzip master.zip

Navigate into the project directory

cd SecureTea-Project

Install SecureTea package

sudo python setup.py install
